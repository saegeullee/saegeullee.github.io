`Bootstrap-Vue` 프로젝트를 진행하기 위해서는 먼저 `vue cli`가 글로벌로 설치되어 있어야 한다. 최초에 설치가 안되어 있다면 다음 명령어로 vue cli를 설치한다.

```
npm install -g @vue/cli
```

그 다음은 vue 프로젝트를 만든다.

```
vue create your-vue-app-name
```

그 다음 bootstrap-vue 를 추가한다.

```
vue add bootstrap-vue
```
